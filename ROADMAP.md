# Family Planner Backend - ROADMAP

## 📋 프로젝트 개요

가족 플래너 시스템을 위한 NestJS 백엔드 REST API입니다.

### 기술 스택
- **Framework**: NestJS v11
- **Language**: TypeScript v5.7
- **Database**: MySQL (Prisma ORM v6.19)
- **Authentication**: JWT (passport-jwt)
- **Testing**: Jest v30
